# Sengoku Website - Index Page 2026 Draft

Working draft for the new subscription-focused index page.

Primary goal:

Convert visitors into mailing list subscribers before crowdfunding.

Secondary goals:

- Make Sengoku immediately understandable.
- Preserve the strong visual identity and artwork.
- Give players, buyers, and publishers clear routes to deeper information.
- Move technical details, rulebook content, component lists, and sell sheet material into dedicated sections/pages.

## 04 - What Is Sengoku?

### Visual Role

Use one strong character artwork or prototype image beside/above a short explanation.

This replaces the long historical introduction and the current dense game description.

### Draft Text

Sengoku is a medium-weight strategy board game for 2-4 players, with an expanded version planned for up to 6.

Players lead rival daimyos, use asymmetric decks, form unstable alliances, and compete for influence across a changing map of feudal Japan.

Easy to learn.

Hard to read.

Cruel when you stop paying attention.

## 05 - The World Of Sengoku

### Visual Role

Use the samurai / historical art here, but reduce the history copy.

### Draft Text

A brutal age. A strategic game.

Set during Japan's Sengoku period, Sengoku turns a century of shifting alliances, ambition, betrayal, and war into a tense strategy game for the table.

## Content To Move Out Of The Main Index Page

These items should not dominate the main subscription landing page.

Move to How to Play / Rulebook:

- Detailed rules
- Turn structure
- How cards work
- Duel variant explanation
- Full manual PDF

Move to Publisher Info / Sell Sheet:

- Full component list
- Target price range
- Manufacturing version notes
- Base vs special edition details
- Player count variants
- Prototype status
- Contact details for publishing/manufacturing

Move to optional Lore / About section:

- Longer historical Sengoku period explanation
- Full design origin story
